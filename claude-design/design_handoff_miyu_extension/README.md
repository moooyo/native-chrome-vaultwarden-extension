# Handoff: 密屿 MiYu — 密码管理器浏览器插件

## Overview
「密屿 MiYu」是一款兼容 Bitwarden / Vaultwarden 后端协议的密码管理器浏览器插件，产品能力对标 1Password 浏览器插件。设计覆盖三大部分：

1. **Popup 弹窗** — 点击工具栏图标弹出：解锁、搜索、分类筛选、当前网站匹配置顶、条目详情（复制/填充/TOTP）、密码生成器、同步状态
2. **自动填充（content script）** — 登录表单旁的侧挂面板：匹配项点击填充、2FA 验证码填充、注册场景内联生成强密码（规则就地可调）、锁定提示、Passkey 选择器
3. **设置页（options page）** — 账户与同步、安全、自动填充、密码生成器、**Send（临时加密分享，兼容 Bitwarden Send）**、外观、导入导出、关于

## About the Design Files
本包中的 `密屿 · 插件设计.dc.html` 及配套 JS 是**用 HTML 制作的设计参考稿（可交互原型）**，展示目标外观与交互行为，**不是可直接上线的生产代码**。任务是：在目标代码库的现有技术栈中（如 React + WebExtension MV3、Vue、Svelte 等）**重新实现这些设计**；若尚无代码库，推荐 WebExtension Manifest V3 + 任意现代前端框架，content script 内的 UI 必须挂载在 Shadow DOM 中以隔离宿主页面样式。

打开 `密屿 · 插件设计.dc.html` 即可在浏览器中交互体验全部状态（需与 `support.js`、`browser-window.jsx` 同目录）。

**重要**：文件内画布分三个 turn 区块。**最终定稿 = 顶部 turn 3 + turn 2**（id 前缀 `3x`/`2x`）。最下方 turn 1（Fluent/WinUI 风格，id `1x`）是已废弃的早期方向，**忽略**。方案 2d（内联胶囊）是未选中的备选，自动填充统一采用 **2c 侧挂面板**样式。

## Fidelity
**High-fidelity（高保真）**。颜色、字体、字号、间距、圆角、阴影、文案均为最终值，应按像素级还原（数值见下文与源码 inline style）。浏览器窗口镜框（标签栏/地址栏，`browser-window.jsx`）仅为演示场景，不属于交付物。

### B. 自动填充 — 侧挂面板样式（定稿 2c；扩展 3a、2e、2f、3b）
面板**不遮挡表单**：挂在输入框/表单卡片右侧，用 7px 苔绿圆点 + 22px 细线与聚焦输入框相连。面板宽 272–280px，白底圆角 14，阴影 `0 16px 40px rgba(20,24,32,.16)`，入场动画：向下 5px 位移淡入 0.18s ease-out。

- **2c 登录匹配**：头部 = 16px mini logo +「密屿」11.5px 苔绿 600 + 右侧「2 个匹配项」10.5px 灰。行 = 30px tile + 标题/用户名 + 「填充」文字按钮（苔绿）。点击 → 表单填入用户名/密码、输入框右上角出现 22px 绿色对勾角标、面板收起变「已填充」胶囊 + 撤销。**登录场景不含"生成密码"入口**
- **3a 2FA 验证码页**：验证码输入页（6 格）同样侧挂面板：匹配条目 + 灰底码框（18px mono 苔绿实时验证码 + 剩余秒数 + 30s 进度条）+「填充验证码」主按钮 + 复制按钮。填充后 6 格显示数字（苔绿描边），面板变「已填充验证码」+ 撤销
- **2e 注册生成**：新密码输入框下方弹卡：建议密码（mono 着色）+ **就地调整规则**：长度滑杆 + `0–9` / `!@#$` 两个胶囊开关（选中苔绿 10% 底描边）+ 换一个按钮 + 元信息（"极强 · 18 字符 · 含符号"）+「使用此密码」墨黑主按钮 + 说明「使用后将自动保存到你的密钥库」。使用后双栏填充 + 「已保存到密屿」确认条 + 撤销
- **2f 锁定态**：**页面内不提供解锁输入**。输入框角标变灰锁 icon，下方信息条：「密屿已锁定 · 2 个匹配项不可见」+「点击右上角工具栏中的密屿图标解锁」

## Interactions & Behavior
- **TOTP**：RFC 6238，30s 周期；UI 每秒更新剩余秒数，进度条 `transition: width 1s linear`；所有出现处（popup 详情、3a 面板）都带复制按钮
- **复制**：写剪贴板 + 底部 toast（深色胶囊 `rgba(22,24,29,.92)`，圆角 16，1.9s 后消失，动画上移淡入 0.18s）；密码/验证码复制提示「30 秒后自动清除」（对应安全设置）
- **同步**：手动按钮 + 解锁/变更时自动同步；同步中 icon `rotate 360° 0.8s linear infinite`
- **解锁**：Enter 提交；生物识别（WebAuthn/平台 API）
- **搜索**：对标题/用户名/域名做 includes 过滤，与分类 chips 组合
- **开关**：40×20（Send 表单内 36×19），knob 14px 白圆 + 微阴影，`transition .15s`；开 = 苔绿底
- **循环下拉**（原型简化）：实现时用真实下拉菜单
- **hover**：列表行 = 浅灰底（`#F2F2EF` / 深色 `rgba(255,255,255,.05)`）；icon 按钮 = `rgba(22,24,29,.06)`；主按钮墨黑 → `#2A2D34`
- **面板入场**：`translateY(-5px)+fade 0.18s ease-out`

## State Management
- 全局：锁定状态、主密码派生密钥（内存）、上次同步时间、同步中标志
- 密钥库：条目数组（类型 login/card/totp/note/identity；字段 title/username/password/domain/totpSecret/passkey 标记）
- Popup：搜索词、选中分类、展开条目 id、密码可见性、当前视图（vault/generator）
- 生成器：length(8–40)/upper/number/symbol/mode(random|words|pin)，规则变更即重新生成；默认值来自设置页
- 自动填充：per-tab 匹配列表、填充完成态、撤销
- Send：列表（名称/类型/过期/访问计数）、创建表单草稿
- 设置：各开关与枚举值，持久化到 `chrome.storage`
- 后端：Bitwarden/Vaultwarden REST API（登录、sync、cipher CRUD、Send CRUD），端到端加密
